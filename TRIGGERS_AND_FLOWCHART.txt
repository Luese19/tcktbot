=================
DIRECT TRIGGERS
=================

/start              -> Detailed ticket form
/register_email     -> Register company email
/help               -> Show all commands
/lookup             -> Search your tickets
/admin              -> Admin panel
/status             -> Bot status

@botname issue      -> Instant group ticket (GROUP ONLY)


=================
WHAT YOU NEED TO DO
=================

FOR GROUP MENTION TO WORK:

1. Run: python bot/main.py
   (Keep terminal open)

2. Add bot to Telegram group

3. In group, type: @mevithelpdeskbot [issue]

4. Wait 5 seconds for ticket


FOR DIRECT MESSAGE FORM:

1. Run: python bot/main.py
   (Keep terminal open)

2. Send /start to bot (private)

3. Fill in the form

4. Ticket created


=================
THE KEY THING
=================

Bot must be RUNNING (python bot/main.py)
If terminal is closed = bot is offline
No messages will be processed


=================
VERIFY BOT IS WORKING
=================

Sign: Bot is online and ready to accept:
- Group mentions: @mevithelpdeskbot issue
- Private commands: /start, /help, etc
- Ticket creation from any trigger

Try this to test:
1. Start bot: python bot/main.py
2. Send /help to bot (direct)
3. Bot responds with help menu
4. If yes = working perfectly
5. Then use group mention feature
